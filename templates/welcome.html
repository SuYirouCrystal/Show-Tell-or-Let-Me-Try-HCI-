COMSE6178 HCI Study Prototype

Show, Tell, or Let Me Try?

Compare how visual, textual, and interactive AI explanations shape product decisions, confidence, and trust calibration.

{% if has_session %} Resume current session {% endif %}

What participants do

  1. Review six product choice scenarios.
  2. See one AI recommendation per task.
  3. Use one assigned explanation format for the full session.
  4. Choose a product, judge the AI, and rate confidence.
Show

Visual Explanation

Feature-importance bars highlight which criteria the recommendation weighted most.

Tell

Text Explanation

Step-by-step language explains the recommendation in the same decision context.

Let Me Try

Interactive Explanation

Sliders let participants adjust priorities and observe simulated recommendation changes.

Research focus

Understanding, trust, and cognitive load

The backend keeps model outputs static across conditions so the study can isolate explanation modality rather than model accuracy. Some AI recommendations are intentionally wrong so the system can measure error detection and calibrated trust.
